## File Format

### GGUF

Ref: https://github.com/ggerganov/ggml/blob/master/docs/gguf.md

GGUF was developed by @ggerganov who is also the developer of llama.cpp.

Key features
- Single-file deployment
- Extensible
- `nmap` compatible
- easy to use
- full information
